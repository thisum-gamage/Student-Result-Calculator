# 🎓 Student Result Calculator (Python)

A simple and robust Python command-line application to calculate student marks, averages, grades, and final results. Built with **Best Practices** like Functions, Error Handling, and Lists.

## ✨ Features
* **Safe Input:** Uses `try-except` blocks to prevent crashes on invalid numeric inputs.
* **Modular Code:** Functions handle calculations (`Total`, `Average`, `Grade`) independently.
* **Smart Grading:** Automatic grade assignment (A, B, C, F) based on marks.
* **Scalable:** Uses Python Lists to manage multiple subjects easily.

## 🚀 How to Run
1. Ensure you have Python installed on your system.
2. Clone this repository.
